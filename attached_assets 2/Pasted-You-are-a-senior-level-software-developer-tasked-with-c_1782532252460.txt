You are a senior-level software developer tasked with creating a personal productivity dashboard website that is both inspiring and intuitive. The UI/UX must evoke daily motivation and a sense of accomplishment, using bright, uplifting colors and subtle 3D effects. Draw inspiration from two key sources: a) ajiteshvishwakarma.vercel.app, which features modular cards and smooth hover transitions; and b) VD-cwd1P9vE, which emphasizes layered scroll effects, parallax depth, and refined micro-interactions.

Key UI/UX Specifications:

Color Palette: Choose a bright, harmonious palette. Primary colors: soft pastel pink (hex: #FFB6C1), lavender (hex: #E6E6FA), and light turquoise (hex: #AFEEEE). Use white or off-white backgrounds to maximize brightness. All text must have strong contrast for readability.
Layout Structure: The dashboard should be a grid of modular cards, each dedicated to a category: daily goals, study hours, financial expenses, and health metrics. Cards should stack vertically but adapt responsively. Each card must be clickable—expanding to show details when selected, but always returning to the dashboard view.
Animations: Implement 3D hover effects—when a user hovers over a card, it slightly elevates, with a gentle shadow and a subtle scaling effect. Use parallax scrolling—when the user scrolls, background layers move at a different pace than the foreground. Transitions between sections should fade or slide smoothly, ensuring no sudden jumps.
Micro-Interactions: When a user completes a task, animate a small celebratory icon (e.g., a checkmark) scaling up. When scrolling to the next section, smoothly transition so that each card slides or fades in. Ensure all animations are lightweight—no lag.
Typography: Use a modern sans-serif font (e.g., Montserrat or Lato). Headings should be large, bold, and spaced well. Body text should be clear and easy to skim. Ensure that the site is fully responsive—animations and layout should adapt smoothly across desktop, tablet, and mobile.
Interactive Elements: Allow each card to be edited—when clicked, it expands into a detailed form where you can add or update progress. Add a sidebar or top navigation bar that remains visible so users can switch between sections seamlessly.
Visual Feedback: When a user completes a habit or task, animate a positive reinforcement—e.g., a confetti pop or a gentle upward progress bar. These small visual cues should give a sense of reward and help keep the user motivated.
